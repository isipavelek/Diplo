Ejecutar el proyecto desde una línea de comando de la siguiente manera:

> python main.py parquet_file

Luego de ejecutar, los gráficos generados se almacenan dentro de directorio del proyecto en la carpeta results.
La ejecución imprime por línea de comando la siguiente secuencia de mensajes:

Seting projects outputs folders
Loading DataSet --> data-ecomm-10-19-smallsampled.parquet
data-ecomm-10-19-smallsampled.parquet --> successfully loaded
Starting general analysis
Performed retargeting/remarketing analysis with clean DataSet
Customer head analysis
Samsung brand analysis
See results on ./results folder
